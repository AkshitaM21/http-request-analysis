# HTTP Request Analysis

This project analyzes the HTTP request/response cycle of Wikipedia using Chrome Developer Tools.

## Repository Contents

- report.md – Detailed analysis report
- network.png – Network tab screenshot
- header.png – Request/Response headers screenshot
- timing.png – Timing breakdown screenshot

## Website Analyzed

https://www.wikipedia.org/

## Short Description

Analyzed Wikipedia's HTTPS request cycle using Chrome DevTools, examining DNS lookup, TCP connection, TLS handshake, headers, and TTFB.
